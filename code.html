Design Principles Applied

  • check_circle

    Tonal Architecture: Using background shifts (#f8fafb to #f2f4f5) instead of borders to define navigation shells.

  • check_circle

    Editorial Authority: Tight tracking-tighter on Manrope headlines for a professional, premium "Pugarch" wordmark.

  • check_circle

    Service Differentiation: Distinct color coding (Primary Green for Forestry, Secondary Blue for Water) while maintaining core layout consistency.

Implementation Note

These headers are designed to sit at the top of a 72-unit width NavigationDrawer. Each variation respects the "No-Line" rule, utilizing subtle elevations and color shifts. The iconography uses Material Symbols Outlined with specific weighting to balance with the heavy Manrope typography.
